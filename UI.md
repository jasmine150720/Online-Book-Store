# UI
## List of Interface
### User
- Homepage
- Register/Login
- Interface for viewing classified books (old/ recent/ new books)
- Interface for book details
- Sell old books
- Edit profile
- Order interface + Shopping cart interface
- View order
- Help center
- Setting
### Admin
- Login
- Admin homepage
- Interface of Orders
- Interface of All Books
- Interface of Add Books
